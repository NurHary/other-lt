# Apa itu bahasa pemograman C
bahasa pemograman C adalah bahasa pemograman General Purpose (GPL) tingkat tinggi dengan tujuan untuk pembuatan software, 
dengan yang paling umum adalah untuk pembuatan Operating system, Embedded system, Compilers, Databases, Game Engine, dan lainnya.

## Sejarah Singkat Bahasa Pemograman C
bahasa pemograman C di ciptakan oleh Dennis Ritchie dan Ken Thompson di Bell Lab pada tahun 1970an dengan tujuan utama sebagai bahasa
pemograman untuk mengembangkan Operating System Unix yang pada saat itu tengah dikembangkan oleh Bell Labs.

Hingga hari ini, bahasa pemograman C masih sangat aktif digunakan oleh para pengembang terutama pada bagian dari suatu program yang
membutuhkan performa tinggi serta penggunaan memori secara efisien.
